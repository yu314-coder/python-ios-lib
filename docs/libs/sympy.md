# SymPy

**Pure Python** | v1.14.0 | Computer Algebra System

> Symbolic mathematics: algebra, calculus, equation solving, matrices, and more. Fully offline.

---

## Core Modules

| Module | Key Functions |
|--------|--------------|
| **Symbols** | `symbols`, `Symbol`, `Rational`, `Integer`, `pi`, `E`, `I`, `oo`, `zoo`, `nan` |
| **Simplification** | `simplify`, `expand`, `factor`, `collect`, `cancel`, `apart`, `together`, `radsimp`, `powsimp`, `trigsimp` |
| **Calculus** | `diff`, `integrate`, `limit`, `series`, `summation`, `product`, `Integral`, `Derivative` |
| **Solvers** | `solve`, `solveset`, `linsolve`, `nonlinsolve`, `dsolve`, `rsolve`, `diophantine` |
| **Matrices** | `Matrix`, `eye`, `zeros`, `ones`, `diag`, `det`, `inv`, `eigenvals`, `eigenvects`, `rref`, `nullspace` |
| **Printing** | `pretty`, `latex`, `mathml`, `ccode`, `fcode`, `jscode` |
| **Logic** | `And`, `Or`, `Not`, `Implies`, `satisfiable`, `simplify_logic` |
| **Sets** | `FiniteSet`, `Interval`, `Union`, `Intersection`, `EmptySet`, `S.Reals`, `S.Integers` |
| **Combinatorics** | `factorial`, `binomial`, `fibonacci`, `bell`, `catalan`, `partition` |
| **Number Theory** | `isprime`, `nextprime`, `factorint`, `divisors`, `totient`, `mobius` |
| **Geometry** | `Point`, `Line`, `Circle`, `Triangle`, `Polygon`, `Segment`, `Ray` |
| **Physics** | `mechanics`, `units`, `hydrogen`, `paulialgebra` |
| **Plotting** | `plot`, `plot3d`, `plot_parametric`, `plot_implicit` (via matplotlib shim) |
